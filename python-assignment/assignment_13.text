1. What advantages do Excel spreadsheets have over CSV spreadsheets?

    Ans-1: Excel spreadsheets support advanced features such as formulas, formatting, charts, multiple sheets, and cell styling, while CSV files are simpler and only store data in a plain text format without these features.

2. What do you pass to csv.reader() and csv.writer() to create reader and writer objects?

    Ans-2: You pass a file object (usually opened in read or write mode) to csv.reader() and csv.writer().

3. What modes do File objects for reader and writer objects need to be opened in?

    Ans-3: Reader objects need to be opened in read mode ('r'), and writer objects need to be opened in write mode ('w').

4. What method takes a list argument and writes it to a CSV file?

    Ans-4: The writerow() method of the csv.writer object takes a list argument and writes it to a CSV file.

5. What do the keyword arguments delimiter and line terminator do?

    Ans-5: The delimiter argument specifies the character used to separate fields in the CSV file (default is a comma), while the line_terminator argument specifies the character or string used to terminate lines (default is a newline character).

6. What function takes a string of JSON data and returns a Python data structure?

    Ans-6: The json.loads() function is used to convert a string of JSON data into a Python data structure (e.g., dict, list).

7. What function takes a Python data structure and returns a string of JSON data?

    Ans-7: The json.dumps() function takes a Python data structure and converts it to a string of JSON data.
